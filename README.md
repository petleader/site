# PET LEADER ACLIMAÇÃO

Página oficial de pré-inauguração da Pet Leader Aclimação.

## Sobre

A Pet Leader Aclimação chega para transformar o conceito de cuidado animal em São Paulo, oferecendo:

- Hotelaria Pet
- Day Care
- Creche
- Banho & Tosa
- Atendimento Veterinário
- Pet Shop Premium

Aclimação — São Paulo/SP
